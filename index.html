Node.js

Node.js is used in Javascript and enables development of fast web servers by bringing event-driven programming to web servers.
A project I have created uses Node.js to create a team profile. This team profile builder is a command-line-input application that uses Node.js to run. It requests information about a software engineering team, and generates an HTML webpage displaying a summary for each team member. Once the application is launched in the command-line, the user is presented with a series of prompts. The user can choose the type of employee they want to input - Manager, Engineer, or Intern. Further prompts request information about the employee including email, ID, and GitHub. The user can input as many employees as they like, and once finished, an HTML page is generated displaying the employees and information from the user. Find the GitHub repo HERE and see how to run the application yourself!

SQL

MySQL is an open-sources, relational, database management system based on SQL - Structured Query Language. It is used for storing and managing data, e-commerce, and applications.
I have used mySQL for a number of projects involving the storing and managing of data. I have created a command-line-input application that allows a business owner to view employees, departments and roles in the company. The user can simply view departments, employees and the roles available in a company, and also update employee roles. It uses Node.js to run, and mysql to store company data. This application utilises CRUD (create, read, update, delete) to handle the data in mySQL. To run the application yourself, follow this link to my GitHub repo, and follow the README for installation instructions.
